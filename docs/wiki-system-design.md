# 系统设计说明

## 系统概述

重庆旅游推荐与管理系统面向普通游客和管理员。游客可以浏览重庆景点、美食、酒店信息，也可以注册登录并发布社区帖子；管理员可以在后台维护旅游内容、审核评论并管理社区帖子。

## 功能模块

- 用户模块：注册、登录、Token 校验、用户资料更新。
- 景点模块：景点列表、景点详情、景点搜索、后台增删改。
- 美食模块：美食列表、分类查询、价格区间搜索、后台增删改。
- 酒店模块：酒店列表、酒店详情、条件搜索、后台增删改。
- 社区模块：帖子发布、帖子查看、帖子更新和删除。
- 管理员模块：管理员登录、统计面板、内容管理。

## 技术选型

- Spring Boot 负责后端 REST API。
- MySQL 负责业务数据持久化。
- HTML/CSS/JavaScript 负责前端交互界面。
- Express 作为本地前端静态资源服务。
- Git 和 GitHub 用于版本控制、代码托管和项目跟踪。
